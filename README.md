# Пошаговый мануал по сайту

Для предупреждения появления дальнейших багов

## Изменение картинки на главной странице
![Main Page Image](/xREADME_images/main_image.png)

В Тине есть коллекция "Выбор праздника".

В зависимости от выбраного праздника меняется картинка на главной странице сайта.

Каждому празднику есть соотвествующее 4 картинки:
- ПК версия русский язык
- ПК версия украинский язык
- Мобильная версия русский язык
- Мобильная версия украинский язык

Чтобы сайт не выдавал ошибку, то всегда должно быть 4 картинки к каждому празднику.

Картинки находяться в Media Maneger -> Папка MAIN

Внутри папки MAIN есть папка PSD. Там находяться шаблоны для фотошопа.

Если ты хочешь изменить картинку:
1. Ты можешь: либо создать её с нуля, либо использовать шаблон
2. Когда ты закончил созданием картинки, то даёшь ей название в точности как в папке. Тип файла картинки так же должен соотвествовать (jpg).

Например: default-mob-uk.jpg

Сайт видит это как каринка которая ставиться когда ничего не празнуется, мобильная версия, украинский язык.
Обрати внимание, что сайт видит украинский язык как - uk.

Другой пример: new-year-pc-ru.jpg

Сайт видит как картинка которая ставиться когда Новый год, ПК версия, русский язык.

## Добавление картинки в Слайдер на главной странице

![Slider main page](/xREADME_images/sales-slider.png)

Слайдер находяться в Тине в коллекции "Слайдер на главной странице - RU" и "Слайдер на главной странице - UA"

Когда у пользователя выбран язык на сайте как русский, то сайт выдает ему слайдеры с коллекции "Слайдер на главной странице - RU". Если язык украинский, то наоборот.

Пользователь на русской версии сайта не видит слайдеры с "Слайдер на главной странице - UA".

Алгоритм добавления нового слайдера:
1. Создание картинки: в Media Maneger -> папка HOMEPAGE-SLIDER есть шаблоны для фотошопа. Ты можешь использовать их для создания картинок.
2. После того как картинки готовы, ты создаешь папку в папке HOMEPAGE-SLIDER куда ты поместишь эти картинки. Советую, чтобы папка имела логичное имя, например, "summerFinish". Так ты понимаешь, что там будут картинки к слайдеру, который соответсвует акции к окончанию лета.
3. Помещаешь туда 4 картинки: 2 картинки ПК и моб версия к русской версии сайта и 2 картинки ПК и моб версия к украинской версии сайта
4. Создаешь новый файл в коллекции "Слайдер на главной странице - RU" с названием акции. Название файла так же должно быть логичным как и название папки. Выбираешь туда картинки из созданной папки. Аналогично делаешь в коллекции "Слайдер на главной странице - UA"
5. Сохраняешь, ждешь 5 минут, нажимаешь CTRL+F5, новый слайдер добавлен.
