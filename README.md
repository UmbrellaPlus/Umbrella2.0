# Пошаговый мануал по сайту
Для избежания багов
## Добавление картинки на главную страницу
![Main Page Image](/xREADME_images/main_image.png)
В Тине есть коллекция "Выбор праздника"
К каждому празднику есть соотвествующее 4 картинки
2 картинки на пк версию (русский и украинский язык) и 2 картинки на мобильную версию (русский и украинский язык).
Картинки находяться в Media Maneger -> Папка MAIN
Внутри папки MAIN есть папка PSD. Там находяться шаблон для фотошопа
Если ты хочешь изменить картинку:
1. Ты можешь: либо создать её с нуля, либо использовать шаблон
2. Когда ты закончил с картинкой то даёшь ей название в точности как в папке. Тип файла картинки так же должен соотвествовать (jpg).
Например: default-mob-ua.jpg
Сайт видит это как каринка которая ставиться когда ничего не празнуется, мобильная версия, украинский язык.
